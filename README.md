# 🚀 BTC-Realtime-Monitor
A high-performance, lightweight Python automation tool for real-time Bitcoin price tracking and trend analysis. Optimized for low-resource environments.

## ✨ Key Features
* Real-time Monitoring: Fetches live BTC/USD data with high frequency.
* Smart Logging: Saves every price movement and action into a persistent trading_log.txt file.
* Trend Detection: Analyzes price percentage changes to identify buy/sell opportunities.
* Low Footprint: Designed to run efficiently on systems with limited resources (e.g., 4GB RAM).
* Docker Ready: Built with containerization in mind for 24/7 VPS deployment.

## 🛠️ Technical Stack
* Language: Python 3.x
* Libraries: requests, datetime, os
* Environment: Cross-platform (Windows, Linux, Docker)

## 📂 Project Structure
- main.py: The core logic of the bot.
- trading_log.txt: Automated logs for price actions.
- Dockerfile: Deployment configuration for cloud clusters.

## 🚀 Getting Started
1. Clone the repository.
2. Run python main.py.
3. Monitor the live logs in your terminal.
